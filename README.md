# 🇮🇳 Gramin Udyam Manch | ग्रामीण उद्यम मंच

> A rural startup investment platform connecting village entrepreneurs 
> across India with investors — in Hindi & English.

## 🌟 What is this?

Gramin Udyam Manch is a bilingual (Hindi + English) web platform that 
helps rural startup founders from Indian villages list their business 
ideas and connect with investors — with admin approval and a simple 
₹100 listing fee model.

## ✨ Features

- 🏠 **Home** — live stats (startups listed, contact requests, revenue)
- 📋 **Browse Startups** — filter by state, category, search by name
- 💡 **Submit Startup** — village founders submit their ideas for review
- 🔒 **Investor Contact** — send contact requests (admin notifies within 24hrs)
- 🛡️ **Admin Panel** — approve/reject submissions, manage listings
- 🇮🇳 **Bilingual UI** — Hindi (Devanagari) + English throughout
- 📱 **Fully Responsive** — works on mobile and desktop

## 🗂️ Categories Supported

Agriculture · Food Processing · Handicrafts · Technology · Health · Education

## 🛠️ Tech Stack

- **Frontend** — Vanilla HTML, CSS, JavaScript (single file)
- **Database** — Supabase (PostgreSQL)
- **Fonts** — Baloo 2, Tiro Devanagari Hindi, DM Sans
- **Hosting** — Any static host (GitHub Pages, Netlify, Vercel)

## 🚀 Setup

1. Clone this repo
2. Create a free [Supabase](https://supabase.com) project
3. Run the SQL schema (see `/sql/schema.sql`)
4. Add your Supabase URL + anon key via the **Connect** button in the app
5. Open `index.html` in browser or deploy to GitHub Pages
